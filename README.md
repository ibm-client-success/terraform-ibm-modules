### Pre-requisites
These module makes use of local-exec provisoners to execute bash scripts locally. Some of the scripts may use the following tools:
- [curl](https://curl.haxx.se/)
- [jq](https://stedolan.github.io/jq/)
- [IBM Cloud CLI plug-ins](https://cloud.ibm.com/docs/cli/reference/ibmcloud?topic=cloud-cli-plug-ins)
